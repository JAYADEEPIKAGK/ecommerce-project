# 🛒 E-Commerce Store (Frontend + Backend)

This is a simple full-stack e-commerce project built using **HTML, CSS, JavaScript, and Node.js (Express)**.

---

##  Features

-  Display products
-  Add to cart
- Remove items from cart
-  Total price calculation
-  Order form (Name, Address, Phone)
-  Order success message

---

##  Project Preview

- Product listing with images
- Cart system
- Checkout form

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository


git clone https://github.com/your-username/ecommerce-project.git

cd ecommerce-project


---

### 2️⃣ Run Backend


cd backend
npm install
node server.js


Server runs on:

http://localhost:5000


---

### 3️⃣ Run Frontend

Open this file in browser:


frontend/index.html
